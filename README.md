# اصلاح فرمت و محتوای مارکدان فایل‌های Jupyter Notebook

استفاده از تگ‌های
HTML
در فایل‌های مارکدان در ابتدا یک روش ساده و منطقی برای داشتن محتوای فارسی به نظر می‌رسد، اما برای نوتبوک‌های
Jupyter
این راهکار چند ایراد کلیدی دارد:
- خوانایی محتوای مارکدان از بین می‌رود
- بعضی سلول‌ها، مخصوصا فرمول‌های ریاضی، به درستی رندر نمی‌شوند
- اعمال استایل خیلی سخت می‌شود

این یک اسکریپت ساده برای تبدیل محتوای
HTML
سلول‌های مارکدان در نوتبوک‌های جوپیتر
به مارکدان و اعمال استایل‌‌های منطقی برای زیبایی بیشتر است.

## پیش‌نیازها

- Python 3.8+
- pip  
```bash
pip install markdownify
```

## نحوه استفاده

این اسکریپت در دو حالت کلی عمل می‌کند:
- ایجاد فایل اصلاح شده (پیشفرض)
- جایگزین کردن فایل ورودی

می‌توانید با آرگومان‌های تعریف شده نحوه کارکرد برنامه را مشخص کنید.

- `input_file` : مسیر فایل ورودی `.ipynb`
- `-o, --output` : مسیر فایل خروجی
- `-r, --replace` : جایگزین کردن فایل اصلی
- `-s, --skip-empty` : رد شدن از سلول‌های مارکدان خالی

## منطق کار

به شکل ساده:
- اسکریپت فایل نوتبوک را خوانده و سلول‌های مارکدان را شناسایی می‌کند
- برای اطمینان، در کیس‌هایی که یک فایل دو مرتبه پردازش می‌شود، wrapper ها و استایل‌های قبلی را پاک می‌کند
- فرمول‌های ریاضی را موقتا جدا می‌کند، به دلیل اینکه markdownify ممکن است آنها را خراب کند
- محتوای HTML را به مارکدان تبدیل می‌کند
- فرمول‌ها را برمی‌گرداند
- بر اساس محتوای هر سلول، استایل مناسب را تولید و اضافه می‌کند
- نوتبوک نهایی را ذخیره می‌کند
